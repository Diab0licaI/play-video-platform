# Play - Video Sharing Platform

A full-stack video sharing platform inspired by YouTube, built using the MERN stack.

## Features

- User Authentication (JWT)
- Video Upload & Management
- Playlists
- Watch History
- Liked Videos
- Subscriptions
- Notifications
- Community Posts
- Dashboard Analytics
- Profile Management
- Search Functionality

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Cloudinary
